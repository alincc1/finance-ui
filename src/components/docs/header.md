## header 头部组件（登录后可看到效果）

### 基础用法

:::demo

```html
<template>
  <div>
    <el-header>
      <rd-header backImg="https://res.oss.zqlian.com/public/0021/0001/back.png" />
    </el-header>
  </div>
</template>
```

:::

### Attributes

| 参数  | 说明  | 类型  | 可选值  | 默认值  |
|:----------|:----------|:----------|:----------|:----------|
| showBack    |  路由元信息,为true时显示返回箭头   | Boolean    |     | false    |
